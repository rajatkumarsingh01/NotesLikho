

# 📝 NotesLikho - A Simple To-Do App

NotesLikho is a lightweight and elegant To-Do/Notes app built with **Jetpack Compose**. Designed for simplicity and productivity, it helps you jot down thoughts, manage daily tasks, and keep your ideas organized.

---

## ✨ Features

- ✅ Add new notes quickly
- 🖊️ Edit existing notes with an intuitive dialog
- 🗑️ Delete notes with a single tap
- 🧠 Expand note content in a pop-up dialog
- 🎨 Beautiful pink-themed UI with background image on card click
- 🚀 Built with **Jetpack Compose**, **MVVM**, and **Material 3**
